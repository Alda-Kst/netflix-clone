# Netflix Clone App
A React web application to search for movies using the [TMDB API](https://www.themoviedb.org/)

## 🚀 Features

- 🔎 Search for movies by title
- ⏳ Loading spinner while fetching data
- 🎨 Responsive and modern UI

## 🛠️ Tech Stack

- React
- TMDB API
- HTML5 + CSS3
- JavaScript (ES6+)
